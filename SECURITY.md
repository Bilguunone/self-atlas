# Security Policy

## Supported versions

Self Atlas is early-stage. Security fixes should target the current `main` branch unless a release branch exists.

## Reporting a vulnerability

Please open a private security advisory or contact the maintainers directly. Do not post private vault contents, raw captures, tokens, or local filesystem paths in a public issue.

## Privacy-sensitive reports

If a bug involves leaked graph exports, source captures, relationship notes, health notes, financial notes, or other private vault material, share the smallest synthetic reproduction possible. A fake demo vault is much better than a real one.
